  Projects & Publication

Implementation of nano-satellite software using FreeRTOS

Centre for Small Satellites Research, Bangalore, India

As part of my Final Year Engineering Project, I took up the project to implement the overall software of the STUDSAT-2 nanosatellite. The implementation consists of writing device drivers for satellite sensors and actuators like Magnetometer, Reaction Wheels, GPS, etc. and designing the tasks/processes for the satellite as per the FreeRTOS kernel. The software was implemented and tested in STM32F4-Discovery Microcontroller Board. The context switching of tasks was monitored using Tracealyzer software.

Environment: C, STM32F4-Discovery Board, FreeRTOS, Tracealyzer, Magnetometer, GPS, Reaction Wheels

Attitude Determination Algorithm using Kalman Filter

Centre for Small Satellites Research, Bangalore, India

I undertook this project as a part of my Engineering Internship Program. In this project, I implemented the algorithm in STM32F4-Discovery Board using C Programming Language. I verified the correctness of the implementation by comparing the results from the software running in the microcontroller with the software simulated in Matlab. The highlight of the project for me was learning about Kalman Filters.
